# HRMS Lite

## Project Overview

HRMS Lite is a lightweight web-based Human Resource Management System built as a full-stack coding assignment.

The application allows an admin to:

- Add new employees
- View all employees
- Delete employees
- Mark daily attendance for employees
- View attendance records
- Filter employee and attendance data

The project focuses on delivering a clean, stable, and functional solution within a limited time scope

---

## Tech Stack Used

### Backend
- Python

- Django REST Framework
- django-filter

- SQLite

### Frontend
- React.js
- Vite

- CSS
